# Tarea MongoDB - Catálogo de Productos


Repositorio con el diseño, scripts y evidencias de la actividad de MongoDB.


## Estructura


- `design.md` : Diseño de la base de datos y esquema.
- `scripts/` : Scripts listos para ejecutar en MongoDB Playground o mongosh.
- `insert_100_products.js`
- `basic_queries.js`
- `filters_queries.js`
- `aggregation_queries.js`
- `docs/analysis.md` : Análisis de los resultados de las agregaciones.
- `docs/capturas/` : Capturas de pantalla (subir aquí las imágenes que tomes en Atlas).


## Cómo ejecutar


1. Abrir MongoDB Atlas → tu cluster → Collections → tiendaOnline → products.
2. Abrir Playgrounds (o usar mongosh conectado a la BD).
3. Copiar y pegar los scripts desde `scripts/` en un Playground y ejecutarlos.
